## What to build

Upload video or audio files into a Project. Managed file storage (copy on import, independent of original). Source Media list UI showing uploaded files with name, duration, file size. User can delete a source from the project.

## Acceptance criteria

- [ ] User can upload a video or audio file into a project
- [ ] Uploaded file is copied to managed storage
- [ ] User sees uploaded files listed with metadata (name, duration, size)
- [ ] User can remove a source from the project
- [ ] API integration tests cover upload and listing
