# Web App with Cloud AI Processing

The tool is a web application with AI processing performed server-side on cloud GPUs, rather than a local desktop app with on-device AI. This decision was driven by the need for access to large, capable AI models (transcription, NL instruction parsing, compression analysis) that are impractical to run on consumer hardware. The trade-offs are: users require internet connectivity; large video files must be uploaded; privacy-sensitive footage cannot be processed locally; and the service incurs ongoing compute costs. We accepted these trade-offs because the AI capabilities required for the core product vision (natural-language editing, intelligent shortening) exceed what current consumer hardware can deliver with acceptable quality and latency.
