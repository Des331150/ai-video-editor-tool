## What to build

Create, list, get, and delete Projects. FastAPI project scaffold with Postgres connection. React + Vite SPA scaffold with a project list page and a "New Project" button. The user can create a project, see it in a list, open it (to an empty editor shell), and delete it.

## Acceptance criteria

- [ ] User can create a project with a name
- [ ] User sees all their projects in a list
- [ ] User can open a project and see the empty editor layout (no content yet)
- [ ] User can delete a project
- [ ] API integration tests cover CRUD

## Blocked by

None - can start immediately.
